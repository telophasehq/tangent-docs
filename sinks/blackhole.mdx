---
title: "Blackhole Sink"
description: "Sink data to"
---

Blackhole sink drops data. It is used for Benchmarking.

- `type: blackhole`
- `default` (bool; if true, used when a processor output does not specify a sink)

Example:
```yaml
sinks:
  bench_blackhole:
    type: blackhole
    default: true
```
