---
title: "File Sink"
description: "Write output to a local file"
---

<ParamField path="type" type="string" required>
This should be set to `file`
</ParamField>

<ParamField path="path" type="string" required>
Filepath to sink data
</ParamField>

**Example**

```yaml
sinks:
  local_file:
    type: file
    path: /tmp/out.ndjson
````
